# راهنمای اجرای پروژه (مخصوص کسی که کدنویسی بلد نیست)

این پروژه با Next.js + Tailwind CSS + Framer Motion ساخته شده و از دو کامپوننت معروف Aceternity UI استفاده می‌کنه:
- **Spotlight**: نور گرادیانت متحرک پس‌زمینه
- **TextGenerateEffect**: متنی که کلمه به کلمه ظاهر میشه

چون این پروژه روی سیستم من (که اینترنت نداره) ساخته شده، باید این مراحل رو روی کامپیوتر خودتون (که اینترنت داره) انجام بدید.

## مرحله ۱: نصب Node.js
اگه از قبل نصب نکردید:
1. به سایت https://nodejs.org برید
2. نسخه LTS (پیشنهاد شده) رو دانلود و نصب کنید
3. برای چک کردن نصب بودنش، در ترمینال/Command Prompt بنویسید:
   ```
   node -v
   ```
   اگه یک شماره ورژن نشون داد (مثلاً v20.x.x)، یعنی نصب شده.

## مرحله ۲: باز کردن پوشه پروژه در ترمینال
1. فایل zip این پروژه رو از حالت فشرده خارج (Extract) کنید
2. وارد پوشه `aceternity-project` بشید
3. در همون پوشه، ترمینال (یا Command Prompt در ویندوز، یا Terminal در مک) رو باز کنید
   - در ویندوز: داخل پوشه راست‌کلیک کنید و "Open in Terminal" رو بزنید
   - در مک: می‌تونید پوشه رو به آیکون Terminal بکشید یا با کلید راست از منوی Services استفاده کنید

## مرحله ۳: نصب کتابخانه‌ها
در ترمینال این دستور رو بنویسید و Enter بزنید:
```
npm install
```
این دستور تمام کتابخانه‌هایی که در فایل package.json نوشته شده (Next.js، Tailwind، Framer Motion و...) رو دانلود و نصب می‌کنه. ممکنه چند دقیقه طول بکشه.

## مرحله ۴: اجرای پروژه
بعد از تمام شدن نصب، این دستور رو بزنید:
```
npm run dev
```
حالا یک آدرس مثل این در ترمینال نشون داده میشه:
```
http://localhost:3000
```
این آدرس رو در مرورگر (Chrome، Firefox و...) باز کنید. باید صفحه‌ای سیاه‌رنگ با یک نور گرادیانت و متن متحرک ببینید.

## چطور تغییرش بدم؟
- متن اصلی صفحه: فایل `app/page.tsx` رو باز کنید و متغیر `heroText` رو عوض کنید
- رنگ نور Spotlight: در همون فایل، مقدار `fill="white"` رو به رنگ دیگه‌ای مثل `fill="#3b82f6"` (آبی) تغییر بدید
- بعد از هر تغییر در فایل‌ها، فقط کافیه فایل رو ذخیره (Save) کنید؛ صفحه مرورگر خودش به‌روز میشه (نیازی به اجرای دوباره دستور نیست)

## اضافه کردن کامپوننت‌های بیشتر از Aceternity UI
1. به آدرس https://ui.aceternity.com برید
2. کامپوننت دلخواه رو پیدا کنید (مثلاً "Background Beams" یا "Card Hover Effect")
3. کد نمونه‌ای که سایت نشون میده رو کپی کنید و در پوشه `components/ui/` به صورت یک فایل جدید (مثلاً `background-beams.tsx`) ذخیره کنید
4. اون کامپوننت رو مثل نمونه‌های موجود (Spotlight و TextGenerateEffect) با `import` در `app/page.tsx` وارد و استفاده کنید

اگه هر مرحله گیر کرد یا پیام خطایی دیدید، متن خطا رو برام بفرستید تا کمکتون کنم.
